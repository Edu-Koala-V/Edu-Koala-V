## funkcjonalności do zrobienia

statystyka użytkownika
zarządzanie klasami i przedmiotami
aktywacja zadań lekcji i testów dla konkretnej klasy / ?ucznia
ustawienia kolorów motywu dla użytkownika
statystyki progresji dla uczniów danej klasy i miedzy klasami ( grywalizacja)
achievements do przemyślenia / implementacji

## Co zrobione

Lekcje da się tworzyć to samo z zadaniami
widok dla lekcji jest już podzielony i gotowy
Użytkownik może mieć własny avatar

Trzeba kontynuować prace nad zadaniami i rozpocząć prace nad testami

Całość zaczyna zamulać trzeba myśleć nad optymalizacją całego systemu (?COOKIES Cache)
